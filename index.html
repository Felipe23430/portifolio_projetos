Projeto de Previsão de Vendas

Desenvolvimento de um modelo preditivo de Machine Learning para prever as vendas diárias. Segui a metodologia CRISP-DS para garantir um ciclo de desenvolvimento estruturado e reprodutível. Realizei análise exploratória de dados (EDA) e validação de hipóteses de negócio, identificando insights como a saturação de promoções de longo prazo e o impacto negativo de um sortimento muito amplo. Executei todo o pipeline de dados: engenharia de features, tratamento de dados, encoding e seleção de features com o algoritmo Boruta.

Resultado de Negócio: O modelo fornece previsões confiáveis que permitem uma alocação de recursos mais eficiente e um planejamento estratégico mais assertivo.

As ferramentas utilizadas foram:

  • Python
  • Jupyter Notebook
  • Machine Learning
  • GitHub

Painel Gerencial com Streamlit

Nesse projeto, os conceitos de Programação em Python, manipulação de dados, pensamento estratégico e lógica de negócio, junto com ferramentas de desenvolvimento web como o Streamlit e GitHub, foram usados para desenvolver um painel gerencial com as principais métricas de uma empresa marketplace de delivery de comida na Índia.

O resultado final do projeto foi um painel hospedado em um ambiente Cloud e disponibilizado através de um link web. O painel pode ser acessado por qualquer dispositivo conectado à internet.

As ferramentas utilizadas foram:

  • Python
  • Jupyter Lab
  • Terminal
  • Streamlit
  • Streamlit Cloud

Ensaio de Machine Learning

Nesse projeto, foram explorados os principais conceitos do treinamento, ajuste de parâmetros e limiares entre overfitting e underfitting de vários algoritmos de Machine Learning dentro das tarefas de classificação, regressão e agrupamento, através de um ensaio de Machine Learning.

O resultado final do projeto foi um painel mostrando a performance desses algoritmos de Machine Learning, a partir da variação dos valores dos principais parâmetros de cada algoritmo, que controlam o estado de overfitting e underfitting.

As ferramentas utilizadas foram:

  • Python 3.10 e Scikit-learn
  • Algoritmos de classificação: KNN, Decision Tree, Random Forest e Logistic Regression
  • Métricas: Accuracy, Precision, Recall, F1-Score, R², MSE, RMSE, MAE, MAPE, Silhouette Score
  • Algoritmos de regressão: Linear, Decision Tree, Random Forest, Lasso, Ridge, Elastic Net
  • Algoritmos de agrupamento: K-Means e Affinity Propagation
  • More
