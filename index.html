Projeto de Previsão de Vendas - Rossmann Stores | Data Science & Machine Learning

Desenvolvimento de um modelo preditivo de Machine Learning para prever as vendas diárias

Segui a metodologia CRISP-DS para garantir um ciclo de desenvolvimento estruturado e reprodutível.Realizei análise exploratória de dados (EDA) e validação de hipóteses de negócio, identificando insights como a saturação de promoções de longo prazo e o impacto negativo de um sortimento muito amplo. Executei todo o pipeline de dados: engenharia de features, tratamento de dados encoding e seleção de features com o algoritmo Boruta. Treinei, avaliei e comparei o desempenho de 5 algoritmos de Machine Learning (Random Forest, XGBoost, Regressão Linear, Lasso e um modelo de média).

Resultado de Negócio: O modelo fornece previsões confiáveis que permitem uma alocação de recursos mais eficiente e um planejamento estratégico mais assertivo.

    Ensaio de Machine Learning

    Nesse projeto, foram explorados os principais conceitos do treinamento, ajuste de parâmetros e limiares entre overfitting e underfitting de vários algoritmos de Machine Learning dentro das tarefas de classificação, regressão e agrupamento, através de um ensaio de Machine Learning.

    O resultado final do projeto foi um painel mostrando a performance desses algoritmos de Machine Learning, a partir da variações dos valores dos principais parâmetros de cada algoritmo, que controlam o estado de overfitting e underfitting.

    • Python 3.10 e Scikit-learn
    • Algoritmos de classificação: KNN, Decision Tree, Random Forest e Logistic Regression
    • Métricas de performance: Accuracy, Precision, Recall e F1-Score, R2, MSE, RMSE, MAE, MAPE e Silhouette Score
    • Algoritmos de regressão: Linear Regression, Decision Tree Regressor, Random Forest Regressor, Polinomial Regression, Linear Regression Lasso, Linear Regression Ridge, Linear Regression Elastic Net, Polinomial Regression Lasso, Polinomial Regression Ridge e Polinomial Regression Elastic Net
    • Algoritmos de agrupamento: K-Means e Affinity Propagation
